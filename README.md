# PM
Проект в учебных целях. Песни берутся из хранилища на телефоне. Плеер разделен на 4 раздела: песни, альбомы, артисты, плейлисты. Музыкальные файлы можно удалить (смахнув влево), сортировать, искать по разделам, отправлять другим людям, добавлять в текущую очередь и в плейлисты. Разделы (песни, альбомы, артисты) формируются автоматически, а плейлист заполняется пользователем, когда он создает новые и сохраняет старые сборники. Отображение композиций имеет два вида - плитка и сетка. Имеется поддержка гарнитуры - старт/стоп, убавить/прибавить громкость, предыдущий/следующий трек.
